# Pi-hole blocklist — Self-Harm / Pro-Self-Injury / Suicide Encouragement
# ----------------------------------------------------------------------
# Purpose : Block dedicated pro-self-harm, self-injury glorification,
#           and suicide-encouragement websites at the DNS level.
# Format  : Plain domain list (one domain per line). Compatible with
#           Pi-hole Adlists, AdGuard Home, dnsmasq, /etc/hosts.
# Updated : 2026-05-03
# License : Public domain / CC0
#
# Companion files in this directory:
#   - regex.txt      (regex rules)
#   - allowlist.txt  (recovery/help whitelist)
#   - README.md      (usage + caveats)
#
# IMPORTANT — Recovery sites are NOT on this list. Sites like
# recoveryourlife.com, 7cups.com, healthyplace.com, thesira.org, and
# selfinjury.bctr.cornell.edu use the same vocabulary but exist to help
# users recover. They are pre-allowlisted in the companion file.
# ======================================================================

# investigation (provisional breach finding 2026), Wikipedia coverage.
